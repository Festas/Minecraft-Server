Frequently Asked Questions

How do I join the server?

+

Simply add mc.festas-builds.com to your server list! Java Edition players use the default port, while Bedrock Edition players should connect on port 19132.

Is the server cross-platform?

+

Yes! We support both Java and Bedrock Edition, so you can play on PC, mobile, Xbox, PlayStation, or Nintendo Switch.

How do I protect my builds?

+

Use the golden shovel to claim land with GriefPrevention! Right-click two corners to create a claim. Your builds are automatically protected from griefing.

What are the server rules?

+

Be respectful, no griefing, no cheating, keep chat clean, and follow staff instructions. See the full rules section above for details.

How do build competitions work?

+

Weekly themed competitions are announced in Discord. Claim a plot with /plots auto, build according to the theme, and winners receive exclusive cosmetic rewards!

How do I unlock cosmetics?

+

Cosmetics are earned through playtime, competitions, and achievements. As you rank up from Newcomer to Legend, you'll unlock new particles, hats, and backpacks!
