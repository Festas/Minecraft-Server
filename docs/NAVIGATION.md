2. **Search by Role:**
   - New users ‚Üí Start with [Getting Started](getting-started/)
   - Administrators ‚Üí Check [Administration](admin/)
   - Developers ‚Üí See [Development](development/)
   - Troubleshooting ‚Üí Visit [Troubleshooting](troubleshooting/)

3. **Use Browser Search:**
   - Press `Ctrl+F` (or `Cmd+F` on Mac)
   - Search for keywords on this page
   - Look through the alphabetical index

4. **Check Related Documents:**
   - Most documents have a "Related Documents" section at the bottom
   - Follow cross-references between documents

5. **Review the Archive:**
   - [Archive](archive/) contains implementation details and historical decisions
   - May contain technical details not in main documentation

6. **Still Can't Find It?**
   - Check if the topic is planned in the [Roadmap](features/roadmap.md)
   - Create a documentation issue using the template in `.github/ISSUE_TEMPLATE/documentation.md`
   - Ask in the Discord server
